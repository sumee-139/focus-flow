# タスク管理画面 ワイヤーフレーム

## 設計コンセプト

### Core Design Principles
1. **Quick Task Creation**: 思考を中断せずに素早くタスクを作成
2. **Clear Priority**: 重要なタスクが一目で分かる表示
3. **Effortless Organization**: 直感的なタスク整理
4. **Focus Integration**: 集中モードとの seamless な連携

### 要素詳細

#### 1.1 タスクカード
- **サイズ**: 横幅100%、高さ80px
- **内容**:
  - ドラッグハンドル（⋮⋮）
  - 統一アイコン（📝）
  - タスク名（太字）
  - 簡単な説明（1行）
  - 見積もり時間
  - 集中開始ボタン
  - アラーム時間表示
- **インタラクション**:
  - タップで詳細表示
  - 集中ボタンで即座に集中モード開始
  - 長押しで素早く編集
  - ドラッグハンドルでタスク順序変更

#### 1.2 クイック作成ボタン
- **位置**: タスクリスト下部
- **サイズ**: 横幅80%、高さ50px
- **インタラクション**:
  - タップで新規作成フォーム表示
  - キーボードショートカット（Ctrl+N）

#### 1.3 デイリーメモ
- **位置**: タスクリスト右上（デスクトップ）、タスクリスト下部（モバイル）
- **サイズ**: 横幅100%、高さ200px
- **内容**:
  - プレーンなMarkdownテキストエリア
  - 自由記述（構造化なし）
  - 見出し、リスト、強調など自由に使用可能
- **インタラクション**:
  - ワンクリックで編集モード
  - 自動保存機能
  - Markdown記法リアルタイムプレビュー
  - キーボードショートカット（Ctrl+Shift+D）

## 7. 状態管理とインタラクション

### 7.1 タスク状態
```
[新規] → [進行中] → [完了]
   ↓        ↓        ↓
[保留] → [再開] → [アーカイブ]
   ↓        ↓        ↓
[削除] ← [復元] ← [削除予約]
```

### 7.2 インタラクション
- **ドラッグ&ドロップ**: ⋮⋮ハンドルでタスクの並び順変更
- **スワイプ**: 完了、編集、削除（モバイル）
- **長押し**: 詳細メニュー表示
- **ダブルクリック**: 素早く編集モード

### 7.3 キーボードショートカット
- **Ctrl+N**: 新規タスク作成
- **Ctrl+F**: 検索モード
- **Ctrl+E**: 選択タスク編集
- **Delete**: 選択タスク削除
- **Space**: 選択タスク完了切り替え
